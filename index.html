About

SOVATとは

小さく、軽く、価値あるソフトウェアを作る。 SOVATは、幸福を作るためのデジタル工房である。

Visual / Non-Visual

視覚だけに頼らず、体験そのものの価値を鍛える。 SOVATは、表現ではなく手触りの核心を作る。

Masterpiece Only

量産しない。拡張しすぎない。 研ぎ澄ませた一作だけを、静かに世へ出す。

Principles

SOVAT 五ヶ条

  • バックエンドサービスを実装しない。
  • 基本プレイ無料のガチャゲーを作らない。
  • ゲームループより、価値の密度を優先する。
  • 小さいソフト、軽いソフトを志向する。
  • マスターピースのみを公開する。
Current Project

Saphiness

波乗りの感覚を、触覚で呼び戻す。 視覚に依存しすぎない、Non-Visual SOVAT。

Saphinessは、スマートフォンの振動機能を主軸とした触覚体験ソフトである。 波、間、集中、反応。 画面より先に、身体へ届く体験を目指す。

Concept Touch / Wave / Haptics
Category Non-Visual SOVAT
Status Current Project
Theme 静かな集中と触覚の波

Body reacts before words.

サーフィンの動きそのものではなく、 波に対して身体が応答する感覚を描く。

Key Visuals

Image Visuals

Character

波を読む者。 海の家で、海とサーファーのあいだに立つ。
